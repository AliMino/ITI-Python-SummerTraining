The name of a variable consists of one or more english character (a-z, A-Z) or an underscore `_` followed by zero or more english characters, underscores `_`, or numbers.

## Examples for valid variable names

- name
- age
- python_version
- now
- value1
- value2
- value_2
- \_val_
- _
- _1
- _2
